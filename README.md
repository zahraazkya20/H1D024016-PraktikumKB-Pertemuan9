### Cara Menjalankan
1. Pastikan Python terinstal.
2. Install matplotlib: `pip install matplotlib`
3. Jalankan: `python main.py`

### Parameter
- Jumlah generasi: 50
- Populasi: 20
- Probabilitas crossover: 0.5
- Probabilitas mutasi: 0.1
- Kapasitas tas: 50

### File
- `main.py` - Program utama
- `inisiasipopulasi.py` - Inisialisasi populasi awal
- `EvaluasiFitness.py` - Fungsi fitness
- `selection.py` - Seleksi orang tua
- `crossover.py` - Operasi crossover
- `mutation.py` - Operasi mutasi
